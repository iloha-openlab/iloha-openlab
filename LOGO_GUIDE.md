# Logo Usage Guide

This guide provides instructions and assets for using the official **ILOHA OpenLab LLC** logo.

## 📄 Usage Policy

The ILOHA OpenLab LLC name and logo are protected trademarks. However, we allow the use of these logo assets by third parties to link to our official website (https://www.iloha-openlab.com/) under the following conditions:

1.  **Permission Required:** You must obtain written permission from ILOHA OpenLab LLC before using our logo on any third-party website or material.
2.  **Purpose:** The logo may only be used to point or link to the official ILOHA OpenLab LLC website.
3.  **No Modification:** You may not modify, distort, or change the colors of the logo assets provided.
4.  **No Endorsement:** Usage of the logo does not imply endorsement, sponsorship, or affiliation with your website or product by ILOHA OpenLab LLC.

## 🎨 Logo Assets

The following logo assets are available in the `public/images/` directory of this repository:

### Standard Logo (Icon)
-   [logo.svg](/public/images/logo.svg) - Vector format (recommended)
-   [logo.png](/public/images/logo.png) - Raster format

### Horizontal Logo (Name and Icon)
-   [logo-horizontal.svg](/public/images/logo-horizontal.svg) - Vector format
-   [logo-horizontal.png](/public/images/logo-horizontal.png) - Raster format (various sizes available: 150px, 500px)

## 📩 Requesting Permission

To request permission to use our logo, please contact us through our official website or via email (if available on the site). Please provide the URL where the logo will be displayed and a brief description of the context.
